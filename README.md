# TammaFood

an appllication build with laravel 5.5
 
Hello Tamma

## Spesification

- PHP >= 7.1.3
- MySQL mahmudeeeek
- OpenSSL PHP Extension
- PDO PHP Extension
- Mbstring PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension
- Ctype PHP Extension
- JSON PHP Extension

## How to Install

- `git clone https://github.com/123mahmud/tammacorp.git`
- `cd tammacorp`
- `composer update`
-  create `.env` file
- `php artisan key:generate`
- `php artisan migrate`
- `php artisan db:seed`
- `vendor/bin/phpunit` to run unit testing case
- `php artisan serve` an application will  listening on `http://localhost:8000`
